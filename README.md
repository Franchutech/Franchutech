### Proyectos Destacados

#### dentify-desktop-mvc
Sistema de gestión de escritorio completo diseñado para clínicas dentales. Desarrollado de forma colaborativa en un equipo ágil de 3 integrantes bajo principios Kanban con GitHub Projects. Implementa el patrón MVC en Java, persistencia relacional con MySQL vía patrón DAO y lógica rigurosa en el backend para la validación de horarios y el control de colisiones en la agenda de citas.

#### bootcamp-remote
Portafolio de analítica avanzada que consolida un programa intensivo de 4 semanas enfocado en el ciclo de vida del dato. Incluye el desarrollo de pipelines de extracción, transformación y carga (ETL) con Pandas y SQLAlchemy, consumo de APIs REST, modelado relacional e integraciones con Power BI y cuadros de mando en Excel, junto con análisis exploratorios (EDA) y modelos predictivos iniciales basados en Scikit-Learn.

#### sql-star-schema-retail
Proyecto analítico completo orientado al ámbito de negocio retail. Demuestra la implementación práctica de técnicas avanzadas de modelado multidimensional mediante un Esquema en Estrella (Star Schema), construcción de consultas SQL complejas para la optimización de almacenamiento y la explotación estructurada de métricas corporativas.

#### Evolución API REST (Spring Boot) (Próximamente)
Proyecto enfocado en el desarrollo de servicios web REST, integración de persistencia relacional (PostgreSQL via JPA) y no relacional (MongoDB), asegurando la calidad del código mediante pruebas unitarias con JUnit.
